## To create and run the app locally
- Clone repository
- Install dependencies
  - **Python Dependencies**
    - Make sure you are in the folder that contains Pipfile
    - Run `pipenv install`
  - **Node Modules**
    - Change directory to the `static` folder
    - Run `npm install`
- Running the app
  - Activate environment
  - Run `python3 app.py`
